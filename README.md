# React + Vite

# PLAN

```
src
├── components
│   ├── Card.js          # Displays individual book items
│   ├── Main.js          # Main component with search and book list
│   └── Modal.js         # Displays book details or additional options
│
├── context
│   └── BooksContext.js  # Context provider for book-related states
│
├── hooks                # (optional) Custom hooks for handling reusable logic
│   └── useFetchBooks.js # Custom hook example for fetching book data
│
├── styles               # (optional) CSS files for styling
│   └── main.css         # Example stylesheet
│
├── utils                # (optional) Helper functions
│   └── api.js           # API request functions, e.g., fetching book data
│
├── App.js               # Main app component, sets up context
└── index.js             # Renders App.js to the DOM


```
